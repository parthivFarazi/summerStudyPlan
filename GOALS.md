# GOALS

## Outcome
Be able to **answer the technical questions at any company I'm eligible for** (new-grad, graduating **Dec 2026**). Application window opens late Jul–Oct 2026; be ready to *start* interviewing by ~September. Interviews in **Python**.

## The Aug 20 pivot
Fall semester starts **Aug 20** → time collapses. So all heavy new-pattern learning is front-loaded into a **summer sprint**; fall is maintenance + mocks while interviewing. (Full day-by-day: `Day-by-Day-Roadmap.md`.)

## Pace floors (the thing the DASHBOARD checks)
- **Summer Sprint (now → Aug 19):** ≥ **2 new problems/day**, ≥ **6 sessions/week**. This requires ~doubling the current ~1/day pace — it's the single most important number.
- **Fall (Aug 20 →):** ≥ 3 short sessions/week + daily-ish due reviews; pull new problems (remaining depth, then Hards) as time allows.

## Scope: 115 problems
66 core patterns → +30 depth → +19 canonical Hards/graph-algorithms. Each mastered **2–3× spaced**, not once. Target by Aug 20: all 66 core + a head start on depth.

## Readiness gates (am I there yet?)
- Solve **4 of 5 random unseen Mediums in ≤ 35 min each, unaided**.
- **Name the pattern within 2–3 minutes** of reading a new problem.
- State **time + space complexity unprompted** and correctly.
- **Recover gracefully when stuck** (say where, propose alternatives).
- Self-score every mock on 4 dims: **communication, problem-solving, technical correctness, testing**.

## 2025–26 reality check (shapes how we practice)
Interviews are moving back **in-person**, AI is **banned** in standard rounds, and interviewers probe **"what does this line do — remove it, what breaks?"** ⇒ memorized solutions backfire; **explaining out loud + from-scratch fluency** matter more than raw count. Narrate everything; treat every rep as interview rehearsal.

## Review schedule
- **Weekly:** snapshot the dashboard (sessions, pace-health, mastery, queue trend); confirm nothing decayed silently.
- **Monthly:** check progress vs. the roadmap day; re-balance if a pattern is lagging.

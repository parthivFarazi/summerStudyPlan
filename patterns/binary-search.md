# Binary Search

**Status:** upcoming (Day 9) · **Mastery: 0/5** · Block A

## In one line
Halve a sorted search space (or an answer range) each step → O(log n).

## Reach for it when
- Input is sorted (or you can sort it cheaply)
- 'Find min/first/last that satisfies…'
- Searching a monotonic answer space (e.g., min eating speed)
- Need better than O(n) on a sorted array
